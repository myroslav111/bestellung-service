# Shortcuts intelliJ

| Shortcut        | Description           |
|-----------------|-----------------------|
| strg + alt + o  | Remove unused imports |
| shift + F6      | Rename (Refactor)     |
| strg + shift + r| Search in path        |
| strg + alt + l  | Reformat code         |
